## Como Executar com Docker

### Executar todos os exercícios

```bash
docker-compose up -d
```

### Acessar cada exercício

- **Exercício 1**: http://localhost:8081
- **Exercício 2**: http://localhost:8082
- **Exercício 3**: http://localhost:8083
- **Exercício 4**: http://localhost:8084

### Parar os containers

```bash
docker-compose down
```
